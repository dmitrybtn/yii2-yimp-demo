Class dmitrybtn\yimp\Yimp

All Classes | Properties | Methods | Constants
Inheritancedmitrybtn\yimp\Yimp » yii\base\BaseObject

Main package class, used to render layout snippets.

To use it in your layout you must firstly register it:

$yimp = new Yimp();
$yimp->register($this);

On initialisation, dmitrybtn\yimp\Yimp::$nav and dmitrybtn\yimp\Yimp::$menu properties are instantiated.

To instantiate dmitrybtn\yimp\Yimp::$nav property, firstly existence of Yii::$app->controller->nav is checked. If Yii::$app->controller->nav property exists and is instance of dmitrybtn\yimp\Navigator, it is imported in dmitrybtn\yimp\Yimp::$nav property. Otherwise it is created via DI container by alias, stored in dmitrybtn\yimp\Yimp::DEFAULT_NAV_ALIAS. If you want to set default navigator (i.e. to keep your left menu while working with third-party modules), you can do the following:

Yii::$container->set(Yimp::DEFAULT_NAV_ALIAS, [
    'class' => '\your\navigator\Class',
    // Your config
])

dmitrybtn\yimp\Yimp::$menu property is instantiated via DI container by alias, stored in dmitrybtn\yimp\Yimp::DEFAULT_MENU_ALIAS. You can override it in same way, as you can do it for dmitrybtn\yimp\Yimp::$nav property.
